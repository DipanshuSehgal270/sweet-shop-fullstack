# ðŸ¥¨ Bikaner Sweets - Frontend

A modern, responsive Single Page Application (SPA) for an e-commerce sweet shop. This interface allows users to browse traditional Indian sweets, manage a shopping cart, and simulate secure payments, while providing a comprehensive dashboard for Administrators to manage inventory.

> **Note:** This frontend interacts with a custom-built **Spring Boot Backend** running on Port 8080.

## ðŸ§  Development Methodology & AI Integration

This project was built using a hybrid development approach, leveraging Generative AI as a productivity multiplier while maintaining strict human oversight on core architectural logic.

### ðŸŽ¨ Frontend: AI-Accelerated Development
The frontend was **architected and generated by AI** to simulate a rapid-prototyping environment. The goal was to instantly visualize backend data without spending excessive time on UI boilerplate.
* **Component Architecture:** AI was used to design the React component hierarchy (Pages vs. Components) and enforce separation of concerns.
* **State Management:** The Context API logic for the Shopping Cart and Authentication state was synthesized by AI to ensure efficient prop-drilling avoidance.
* **Styling & UX:** The "SaaS-style" CSS, responsive layouts, and interactive elements (hover effects, payment modals) were generated to achieve a professional aesthetic immediately.

### âš™ï¸ Backend: Human-Led Engineering with AI Support
The Spring Boot backend was **designed, coded, and secured manually** by the developer, with AI serving as an intelligent assistant for specific tasks:
* **Test Driven Development (TDD):** AI was utilized to generate comprehensive **JUnit 5 and Mockito test cases**, ensuring high code coverage for Services and Controllers.
* **Boilerplate Reduction:** AI assisted in scaffolding the initial project structure and generating standard CRUD boilerplate code.
* **Query Optimization:** Complex JPQL and Custom SQL queries (e.g., for the Search/Filtering logic) were refined with AI suggestions for performance.
* **Security Configuration:** AI assisted in debugging specific CORS and CSRF configurations within the Spring Security 6 filter chain.

## âš™ï¸ Installation & Setup

### Prerequisites
* Node.js (v18 or higher)
* The **Bikaner Sweets Backend** must be running on `http://localhost:8080`.

### Steps
1.  **Navigate to the folder:**
    ```bash
    cd frontend
    ```

2.  **Install Dependencies:**
    ```bash
    npm install
    ```

3.  **Run the Development Server:**
    ```bash
    npm run dev
    ```

4.  **Access the App:**
    Open your browser to `http://localhost:5173` (or the port shown in your terminal).

Admin: Logs in -> Sees Edit/Delete buttons on cards -> Can add new inventory via "+ Add Item".
